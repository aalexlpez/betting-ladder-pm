# Landing Page Brief for Codex

## Goal

Build a professional, non-generic landing page for Prediction Ladder that can stand against current prediction-market trading terminal competitors.

## Product

Prediction Ladder is a Windows desktop trading terminal for prediction-market traders. It is desktop-first, provider-aware for Polymarket and Kalshi, live-ready, local/no-custody oriented, and protected by legal/risk/geo/credential/audit gates before live execution.

## Non-negotiables

- Do not make profit claims.
- Do not claim legal certainty.
- Do not imply geoblock evasion.
- Do not create fake testimonials or fake customer logos.
- Do not implement trading logic in the landing app.
- Do not store or reference real credentials.
- Do not claim a Windows download exists unless package output exists.

## Definition of done

The page is done when it feels like a credible trading-terminal landing page, not a generic AI SaaS page, and the evaluator can understand the product, wedge, safety posture, and next action within one minute.
