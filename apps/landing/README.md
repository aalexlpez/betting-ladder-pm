# Landing App

Vite + React + TypeScript static landing page for Prediction Ladder.

This app is the public GTM surface, not a trading surface. It should communicate:

- Windows desktop trading terminal.
- Ladder-first manual execution.
- Live market data.
- Local/no-custody posture.
- Risk-gated live execution.
- Legal and jurisdictional caution.

Before implementing, read Route I and the landing docs in `docs/landing/`.

Use skills:

- `landing-conversion-strategist`
- `frontend-design`
- `web-interface-guidelines-review` after implementation

Do not implement live trading logic, credentials, private keys, or SDK execution paths in this app.
