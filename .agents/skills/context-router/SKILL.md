# Context Router Skill

Use this skill at the start of each Codex task.

Steps:

1. Read `docs/ai/CONTEXT_INDEX.md`.
2. Identify the task route.
3. Read only the route-specific docs.
4. State which route you selected and why.
5. Avoid loading unrelated historical reports.
6. End by updating `docs/ai/context-handoff.md`.
