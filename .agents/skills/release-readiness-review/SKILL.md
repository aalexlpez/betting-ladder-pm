# Release Readiness Review Skill

Use this before demos or end-of-day reports.

Review:

- product readiness;
- typecheck/lint/test/build;
- live gate status;
- legal gate status;
- known issues;
- demo script;
- next steps;
- context handoff.

Output an explicit decision:

- ready for live smoke;
- ready for live-dry-run only;
- ready for paper + real data only;
- not ready.
